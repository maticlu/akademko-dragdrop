#	evaQode Akademko dragdrop

Drag and drop list elements.

**Example:**

HTML:

```html
<ul class="list">
  <li>Example 1</li>
  <li>Example 2</li>
  <li>Example 3</li>
</ul>
```

JS:

```javascript
import EQDragDrop from './evaqode-dragdrop/dragdrop';

const listElement = document.querySelector('.list');
const instance = new EQDragDrop(listElement);
```
